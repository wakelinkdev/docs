# Android App

Пробуждайте свои устройства из любого места с помощью приложения WakeLink для Android.

## Установка

### Google Play Store

[![Get it on Google Play](https://play.google.com/intl/en_us/badges/static/images/badges/en_badge_web_generic.png)](https://play.google.com/store/apps/details?id=io.wakelink)

### Загрузка APK

Скачайте с [GitHub Releases](https://github.com/wakelinkdev/wakelink-android/releases)

### Требования

- Android 8.0 (API 26) или выше
- Подключение к интернету

---

## Начало работы

### 1. Вход в аккаунт

1. Откройте приложение
2. Нажмите **Войти**
3. Введите email и пароль
4. Или нажмите **Создать аккаунт**, если вы новый пользователь

### 2. Добавление первого устройства

Если вы уже зарегистрировали устройства через CLI или веб-интерфейс:

1. Ваши устройства отобразятся автоматически после входа
2. Потяните вниз для обновления списка

Чтобы добавить новое устройство:

1. Нажмите кнопку **+**
2. Введите имя устройства
3. Скопируйте сгенерированный токен
4. Используйте этот токен при настройке вашего ESP-устройства

### 3. Пробуждение устройства

1. Нажмите на устройство в списке
2. Нажмите большую кнопку **Wake**
3. Дождитесь подтверждения

## Возможности

### Список устройств

- Все устройства на одном экране
- Индикаторы статуса онлайн/офлайн
- Метки времени последнего подключения
- Обновление потягиванием вниз

### Быстрые действия

- Нажать на устройство → Немедленное пробуждение
- Долгое нажатие → Дополнительные параметры
- Смахнуть влево → Быстрое пробуждение
- Смахнуть вправо → Просмотр подробностей

### Виджет на главном экране

Добавьте виджет для мгновенного доступа:

1. Долгое нажатие на главном экране
2. Выберите **Виджеты**
3. Найдите **WakeLink**
4. Выберите размер виджета:
   - **Маленький**: Одно устройство, пробуждение одним нажатием
   - **Большой**: Список нескольких устройств

### Уведомления

Получайте уведомления о:

- Результатах команды пробуждения (успех/неудача)
- Изменениях статуса устройства
- Бета-обновлениях

Настройте в разделе **Настройки → Уведомления**

## Настройки приложения

Доступ через значок шестерёнки или меню **Настройки**.

### Аккаунт

- **Email**: Ваш зарегистрированный email
- **Выйти**: Выход из приложения
- **Удалить аккаунт**: Удалить все данные

### Сервер

- **API-сервер**: Стандартный или собственный URL сервера
- **Собственный сервер**: Введите URL вашего self-hosted сервера

### Оформление

- **Тема**: Светлая, Тёмная, Системная
- **Язык**: English, Русский (определяется автоматически)

### Уведомления

- **Результаты пробуждения**: Показывать уведомления о результатах
- **Фоновые обновления**: Периодически проверять статус устройства
- **Звук**: Звук уведомлений вкл/выкл

### Дополнительно

- **Таймаут**: Таймаут команды (по умолчанию 30 с)
- **Режим отладки**: Показывать подробные журналы

## Использование собственного сервера

Подключение к вашему self-hosted серверу WakeLink:

1. Откройте **Настройки**
2. Нажмите **Сервер**
3. Включите **Собственный сервер**
4. Введите URL вашего сервера:
   ```
   https://api.your-domain.com
   ```
5. Нажмите **Проверить соединение**
6. В случае успеха нажмите **Сохранить**

!!! note "Требуется HTTPS"
    Android требует HTTPS для сетевых подключений. Убедитесь, что на вашем сервере настроены действующие TLS-сертификаты.

## Виджеты

### Виджет для одного устройства

- Отображает одно устройство
- Индикатор статуса (онлайн/офлайн)
- Пробуждение одним нажатием
- Обновляется каждые 15 минут

### Виджет для нескольких устройств

- Отображает до 4 устройств
- Индикаторы статуса для каждого
- Нажмите на любое устройство для пробуждения
- Компактный вид

### Настройка виджетов

1. Добавьте виджет на главный экран
2. Выберите устройство(-а) для отображения
3. Выберите частоту обновления
4. При необходимости включите автопробуждение по нажатию

## Устранение неполадок

### «Connection Failed» (Ошибка соединения)

1. Проверьте подключение к интернету
2. Убедитесь, что URL сервера указан верно
3. Попробуйте переключить WiFi/мобильные данные
4. Проверьте доступность сервера: откройте URL API в браузере

### «Authentication Error» (Ошибка аутентификации)

1. Выйдите и войдите снова
2. Проверьте, не изменился ли пароль
3. Подтвердите аккаунт в веб-панели управления

### «Device Offline» (Устройство офлайн)

1. Убедитесь, что ESP-устройство включено
2. Проверьте наличие WiFi-подключения у ESP
3. Проверьте последовательный журнал ESP
4. Возможно, устройство требует перезапуска

### Сбои приложения

1. Очистите кэш приложения: Настройки → Приложения → WakeLink → Очистить кэш
2. Переустановите приложение
3. Сообщите о проблеме на GitHub, приложив журналы сбоев

### Виджет не обновляется

1. Убедитесь, что оптимизация батареи для WakeLink отключена
2. Виджеты обновляются минимум каждые 15 минут
3. Нажмите на виджет для принудительного обновления

---

## Оптимизация батареи

Для надёжных фоновых обновлений:

1. Перейдите в **Настройки → Приложения → WakeLink**
2. Нажмите **Батарея**
3. Выберите **Без ограничений** или **Не оптимизировать**

Это обеспечивает корректную работу виджетов и уведомлений.

## Конфиденциальность

Приложение WakeLink:

- ✅ Хранит учётные данные в защищённом хранилище Android Keystore
- ✅ Использует HTTPS для всех соединений
- ✅ Не отслеживает использование и не собирает аналитику
- ✅ Не обращается к контактам, камере или другим разрешениям
- ❌ Не продаёт и не передаёт ваши данные третьим лицам

Необходимые разрешения:

| Разрешение | Причина |
|------------|---------|
| Интернет | Подключение к серверу WakeLink |
| Фоновый сервис | Фоновое обновление статуса |
