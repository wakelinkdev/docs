# Обзор прошивки

Прошивка WakeLink работает на микроконтроллерах ESP8266 и ESP32, обеспечивая безопасный Wake-on-LAN через интернет.

## Возможности

- 🔐 **Сквозное шифрование** — XChaCha20-Poly1305 AEAD
- 📡 **Постоянный WebSocket** — всегда подключён, мгновенное пробуждение
- 🔄 **OTA-обновления** — обновление прошивки без физического доступа
- 🏠 **Локальный режим** — работает без сервера в локальной сети
- 💤 **Энергоэффективность** — поддержка глубокого сна для работы от батареи
- 🔧 **Веб-настройка** — простая настройка WiFi через captive portal

## Поддерживаемое оборудование

| Плата | Статус | Примечания |
|-------|--------|------------|
| ESP8266 (NodeMCU) | ✅ Рекомендуется | Самая низкая стоимость, проверенная стабильность |
| ESP8266 (Wemos D1 Mini) | ✅ Поддерживается | Компактный форм-фактор |
| ESP32 | ✅ Поддерживается | Больше RAM/Flash, быстрее |
| ESP32-S2 | ✅ Поддерживается | Поддержка USB OTG |
| ESP32-C3 | ✅ Поддерживается | На базе RISC-V |

## Коды состояния светодиода

| Паттерн | Значение |
|---------|----------|
| Медленное мигание (1 Гц) | Подключение к WiFi |
| Быстрое мигание (4 Гц) | Подключение к серверу |
| Горит постоянно | Подключено и готово |
| Двойное мигание | Получена команда |
| Тройное мигание | Ошибка (проверьте логи) |
| Выключен | Глубокий сон или нет питания |

## Потребление энергии

| Состояние | ESP8266 | ESP32 |
|-----------|---------|-------|
| Активный (WiFi TX) | ~170 мА | ~240 мА |
| Активный (ожидание) | ~70 мА | ~80 мА |
| Лёгкий сон | ~1 мА | ~0,8 мА |
| Глубокий сон | ~20 мкА | ~10 мкА |

!!! tip "Работа от батареи"
    Для установок с питанием от батареи:
    
    1. Включите глубокий сон в настройках
    2. Задайте интервал пробуждения (например, проверка каждые 5 мин)
    3. Используйте аккумулятор 18650 + зарядное устройство TP4056
    4. Ожидаемый срок службы: ~30 дней при ёмкости 3000 мАч

## Использование памяти

| Ресурс | ESP8266 | ESP32 |
|--------|---------|-------|
| Flash (прошивка) | ~450 КБ | ~600 КБ |
| Flash (конфигурация) | 4 КБ | 4 КБ |
| RAM (свободная куча) | ~25 КБ | ~200 КБ |
