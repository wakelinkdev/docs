# Справочник кодов ошибок

Полный список кодов ошибок WakeLink и их значений.

## Ошибки протокола (E0xx)

| Код | Название | Описание | Решение |
|------|------|-------------|----------|
| E001 | `INVALID_PACKET` | Некорректная структура пакета | Проверьте формат пакета |
| E002 | `AUTH_FAILED` | Ошибка аутентификации | Проверьте токен, войдите снова |
| E003 | `SEQ_OUT_OF_ORDER` | Неверный порядковый номер | Переподключите сессию |
| E004 | `CHAIN_BROKEN` | Несоответствие хэша цепочки | Сбросьте состояние цепочки |
| E005 | `SIG_INVALID` | Ошибка проверки подписи | Проверьте ключ, переподключитесь |
| E006 | `TS_EXPIRED` | Временна́я метка вне допустимого диапазона | Синхронизируйте время на устройстве |
| E007 | `UNKNOWN_CMD` | Нераспознанная команда | Обновите прошивку |
| E008 | `CMD_FAILED` | Ошибка выполнения команды | Смотрите конкретную ошибку |
| E009 | `DECRYPT_FAILED` | Ошибка дешифрования | Неверный ключ, переподключитесь |
| E010 | `PACKET_TOO_LARGE` | Пакет превышает допустимый размер | Уменьшите полезную нагрузку |

## Сетевые ошибки (E1xx)

| Код | Название | Описание | Решение |
|------|------|-------------|----------|
| E101 | `CONNECT_FAILED` | Невозможно подключиться к серверу | Проверьте сеть, URL сервера |
| E102 | `DNS_FAILED` | Ошибка разрешения DNS | Проверьте настройки DNS |
| E103 | `TLS_HANDSHAKE` | Ошибка TLS-рукопожатия | Проверьте сертификаты |
| E104 | `TIMEOUT` | Тайм-аут соединения | Повторите попытку, проверьте сеть |
| E105 | `DISCONNECTED` | Соединение потеряно | Автоматическое переподключение |
| E106 | `WIFI_DISCONNECTED` | Соединение с WiFi потеряно | Проверьте учётные данные WiFi |

## Ошибки устройства (E2xx)

| Код | Название | Описание | Решение |
|------|------|-------------|----------|
| E201 | `DEVICE_NOT_FOUND` | Устройство не существует | Проверьте имя/ID устройства |
| E202 | `DEVICE_OFFLINE` | Устройство отключено | Дождитесь переподключения |
| E203 | `DEVICE_BUSY` | Устройство выполняет другую команду | Повторите позже |
| E204 | `INVALID_CONFIG` | Недействительная конфигурация | Исправьте конфигурацию |
| E205 | `FLASH_ERROR` | Ошибка флэш-памяти | Сброс к заводским настройкам |
| E206 | `OTA_FAILED` | Ошибка обновления прошивки | Повторите попытку, прошейте через USB |

## Ошибки пробуждения (E3xx)

| Код | Название | Описание | Решение |
|------|------|-------------|----------|
| E301 | `INVALID_MAC` | Неверный формат MAC-адреса | Проверьте формат MAC |
| E302 | `WOL_SEND_FAILED` | Не удалось отправить WoL-пакет | Проверьте сеть |
| E303 | `TARGET_NOT_FOUND` | Цель отсутствует в конфигурации | Добавьте цель |
| E304 | `RATE_LIMITED` | Слишком много команд пробуждения | Подождите и повторите |

## Ошибки сервера (E4xx)

| Код | Название | Описание | Решение |
|------|------|-------------|----------|
| E401 | `UNAUTHORIZED` | Не аутентифицирован | Сначала выполните вход |
| E402 | `FORBIDDEN` | Нет доступа к ресурсу | Проверьте права доступа |
| E403 | `NOT_FOUND` | Ресурс не найден | Проверьте URL/ID |
| E404 | `VALIDATION_ERROR` | Недействительные данные запроса | Исправьте тело запроса |
| E405 | `RATE_LIMITED` | Превышен лимит запросов | Замедлите частоту запросов |
| E406 | `SERVER_ERROR` | Внутренняя ошибка сервера | Обратитесь в поддержку |
| E407 | `SERVICE_UNAVAILABLE` | Сервер временно недоступен | Повторите позже |

## Обработка ошибок

### В коде

```python
try:
    client.wake("my-device")
except WakeLinkError as e:
    if e.code == "E202":  # DEVICE_OFFLINE
        print("Device is offline, queued for later")
    elif e.code == "E304":  # RATE_LIMITED
        time.sleep(60)
        client.wake("my-device")  # Retry
    else:
        raise
```

### В CLI

```bash
if ! wakelink wake my-device; then
    echo "Wake failed"
    exit 1
fi
```

## Типичные проблемы

### E004: CHAIN_BROKEN

**Причины:**
- Устройство или сервер перезапущен, но другая сторона имеет старое состояние цепочки
- Потеря пакетов в сети

**Решение:**
Обе стороны автоматически пытаются восстановить цепочку. При постоянной проблеме — перезапустите устройство.

### E006: TS_EXPIRED

**Причины:**
- Часы устройства значительно отстают
- Задержка сети превышает допустимый порог

**Решение:**
1. Проверьте синхронизацию времени на устройстве
2. Убедитесь в доступности NTP-сервера
3. Увеличьте допуск временно́й метки (конфигурация сервера)

### E202: DEVICE_OFFLINE

**Причины:**
- Устройство обесточено
- Потеряно соединение с WiFi
- Соединение с сервером прервано

**Решение:**
1. Проверьте питание устройства
2. Проверьте подключение к WiFi
3. Просмотрите логи с последовательного порта
4. Перезапустите устройство

### E304: RATE_LIMITED

**Причины:**
- Слишком много команд пробуждения за короткий период
- По умолчанию: 30 команд/минуту

**Решение:**
1. Дождитесь сброса лимита
2. Уменьшите частоту команд
3. Запросите увеличение лимита (при самостоятельном хостинге)
