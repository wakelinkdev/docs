## [1.0.0] - 2024-XX-XX

### Обзор
Первый стабильный релиз экосистемы WakeLink.

### Прошивка (v1.0.0)
- Поддержка ESP8266/ESP32
- Реализация протокола EWSP
- OTA-обновления через защищённый канал
- Веб-режим настройки (provisioning)
- Режим работы в локальной сети
- Генерация WoL-пакетов
- Индикация состояния через LED
- Поддержка глубокого сна

### Сервер (v1.0.0)
- Релейный сервер на базе FastAPI
- WebSocket-соединения (EWSP)
- REST API для управления
- JWT-аутентификация
- Хранение данных в PostgreSQL
- Redis для сессий
- Развёртывание через Docker
- Метрики Prometheus
- Миграции Alembic

### CLI (v1.0.0)
- Обнаружение и сопряжение устройств
- Команды пробуждения со статусом
- Управление устройствами
- Управление конфигурацией
- Несколько форматов вывода
- Автодополнение для оболочки

### Android-приложение (v1.0.0)
- Kotlin Multiplatform (KMP)
- Интерфейс управления устройствами
- Виджеты для пробуждения одним касанием
- Push-уведомления
- Material Design 3
- Биометрическая аутентификация

### Безопасность
- Сквозное шифрование (EWSP)
- XChaCha20-Poly1305 AEAD
- HMAC-SHA256 аутентификация
- Выведение ключей HKDF
- Защита от атак повторного воспроизведения
- Цепочное хэширование для целостности данных

## Схема версионирования

WakeLink следует [Семантическому версионированию](https://semver.org/):

- **MAJOR**: Несовместимые изменения API/протокола
- **MINOR**: Новые функции, обратная совместимость
- **PATCH**: Исправления ошибок, патчи безопасности

### Версионирование компонентов

Каждый компонент имеет независимую версию:
- Прошивка: `wakelink-firmware/VERSION`
- Сервер: `wakelink-server/pyproject.toml`
- CLI: `wakelink-client/pyproject.toml`
- Android: `wakelink-android/app/build.gradle.kts`

### Версия протокола EWSP

Версия протокола согласовывается в процессе рукопожатия:
- Текущая: **v1.0**
- Минимально поддерживаемая: v1.0

## Руководства по обновлению

### Обновление прошивки

```bash
# OTA (рекомендуется)
wakelink ota --device <name>

# Ручная прошивка
esptool.py write_flash 0x0 firmware.bin
```

### Обновление сервера

```bash
docker-compose pull
docker-compose up -d
docker-compose exec server alembic upgrade head
```

### Обновление CLI

```bash
pip install --upgrade wakelink-cli
```
